# 抖音用户主页地址库
# 每行一个链接，以 # 开头的行为注释
# 格式：https://www.douyin.com/user/用户ID
# 
# 🆕 NEW! 现在支持数量限制功能：
# 格式1：https://www.douyin.com/user/用户ID         (不限制，处理所有视频)
# 格式2：https://www.douyin.com/user/用户ID @30     (限制30个视频)
# 格式3：https://www.douyin.com/user/用户ID @10     (限制10个视频)
# 
# 使用说明：
# 1. 将要批量处理的抖音用户主页链接添加到下面
# 2. 每行一个链接
# 3. 在链接后加 @数字 可以限制处理的视频数量
# 4. 数量限制对每个账户独立生效
# 5. 程序会轮流让不同账户访问这些主页，进行点赞和评论
